# Инструкция по установке плагинов Jiko Bridge

## 3ds Max
1. Скопируйте файлы из папки `plugins/3dsmax/` в директорию скриптов 3ds Max (например, `C:\Program Files\Autodesk\3ds Max <версия>\scripts\`).
2. Откройте 3ds Max и выполните скрипт `JikoBridge.ms` через MAXScript.
3. Для Python-скриптов используйте меню Scripting > Run Script и выберите нужный `.py` файл.

## Blender
1. Откройте Blender.
2. Перейдите в меню `Edit > Preferences > Add-ons > Install`.
3. Выберите папку `plugins/blender/addons/JikoBridgeBlend` (или создайте архив и выберите его).
4. Активируйте аддон в списке.

## Cinema 4D
1. Скопируйте содержимое папки `plugins/c4d/` в директорию плагинов Cinema 4D (например, `C:\Program Files\Maxon Cinema 4D <версия>\plugins\`).
2. Перезапустите Cinema 4D.
3. Плагин появится в меню Cinema 4D.

## Houdini
1. Скопируйте файлы из папки `plugins/houdini/` в директорию скриптов Houdini (например, `C:\Users\<user>\Documents\houdini<версия>\scripts\`).
2. Для установки панели используйте файл `toolbar/JikoBridge.shelf`.
3. Перезапустите Houdini.

## Maya
1. Скопируйте файлы из папки `plugins/maya/` в директорию скриптов Maya (например, `C:\Users\<user>\Documents\maya\scripts\`).
2. Откройте Maya и выполните скрипт `JikoBridgeMaya.py` через Script Editor.
3. Плагин появится в меню Maya.
